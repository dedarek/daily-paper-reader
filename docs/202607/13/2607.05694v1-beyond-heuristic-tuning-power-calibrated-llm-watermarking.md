## 摘要
基于logit的水印是一种广泛用于识别LLM生成内容的机制，但其有效性受限于可检测性与语义失真之间的基本权衡。现有分析为超参数的选择提供的指导有限，使得实际部署依赖于启发式调优。在这项工作中，我们开发了一个功率校准的统计框架，建立了水印超参数、检测能力和失真之间的显式定量关系。这种描述将水印设计转化为一个引导优化问题。基于这些结果，我们推导了实用的参数选择程序，在约束条件下实现了最优权衡。跨多个语言模型和数据集的广泛实验验证了该理论，并表明所提出的框架能够一致地识别帕累托最优点。

## Abstract
Logit-based watermarking is a widely used mechanism for identifying LLM generated content, yet its effectiveness is governed by a fundamental trade-off between detectability and semantic distortion. Existing analyses provide limited guidance for principled hyperparameter selection, leaving practical deployments reliant on heuristic tuning. In this work, we develop a power-calibrated statistical framework that establishes explicit quantitative relationships between watermark hyperparameters, detection power, and distortion. This characterization transforms watermark design into a guided optimization problem. Building on these results, we derive practical parameter selection procedures that achieve optimal tradeoffs under constraints. Extensive experiments across multiple language models and datasets validate the theory and demonstrate that the proposed framework consistently identifies Pareto-optimal points.

## 2. 方法论

### 核心思想
- 在标准 KGW 水印框架下，将检测问题形式化为假设检验：
  - H0: 文本未被水印 → green 令牌概率为 γ。
  - H1: 文本已被水印 → green 令牌概率提升为 γ′。
- 通过概率论推导出 γ′ 的显式表达式（定理 3.5），并在大词汇量下给出近似闭合解：  
  \[
  \gamma' = \frac{e^\delta \gamma}{1 + \gamma(e^\delta - 1)} + O(|V|^{-1/2})
  \]
- 利用中心极限定理和弱依赖条件（信息衰减假设 3.6），得到序列级统计量的渐近分布，从而将检测功效表达为：
  \[
  \pi^*(\gamma, \delta) \approx \Phi\left( \frac{\sqrt{n}(\gamma' - \gamma) - z_{1-\alpha}\sqrt{\gamma(1-\gamma)}}{\sqrt{c\gamma'(1-\gamma')}} \right)
  \]
- 失真度量采用逐令牌 KL 散度，推导出闭合形式（引理 3.9）：
  \[
  D_{\text{KL}}(\gamma, \delta) = \delta \gamma' - \log(1 + \gamma(e^\delta - 1))
  \]
  并证明 KL 对 δ 严格单调，对 γ 非单调。

### 关键技术细节
- **绿色列表假设**：伪随机生成，与模型无关，令牌指示器在 H0 下为 i.i.d. Bernoulli(γ)。
- **令牌级替代**：利用 Dirichlet(1,…,1) 先验近似 NTP 分布，推导 γ′。
- **序列依赖处理**：引入信息衰减假设（指数速度）和几何 α-混合条件，证明 CLT 仍成立，方差膨胀常数 c 反映了实际依赖程度。
- **参数优化**：给定 KL 预算 K0 下最大化功效，等价于求解一维问题（由于 KL-δ 的一一对应）。分析发现最优 γ 位于 γ∗ (由功率 >0.5 条件导出) 附近，并推荐使用较大的根 γh。
