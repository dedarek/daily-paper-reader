## 摘要
大型语言模型日益成为敏感数据、不可信输入与特权操作之间交互的中介（尤其在智能体系统中），由此引发安全与隐私风险。这些风险涵盖通过提示注入操控下游工具使用，以及模型输出泄露机密信息。近期基于信息流控制的防御方案展现出潜力，但缺乏推理模型内部信息流的严格语义基础。由于自回归LLM中任意输入令牌可能影响任意输出令牌，现有方法面临严重的污点爆炸问题。

我们提出几何信息流（GIF）——一个从输入令牌到输出追踪信息流的语义框架。GIF利用LLM雅可比矩阵与局部输出几何结构，对扰动输入片段与模型输出之间的香农互信息给出上限，形成可通过自动微分与低秩近似在大模型上计算的可扩展度量。与基于注意力或相关性归因的启发式方法不同，GIF满足局部几何可靠性，我们提供了完全机器验证的Lean 4证明：在局部正则性假设下，GIF上界限定了给定提示所引发的真实信息流。

我们在多个提示注入与隐私泄露基准测试中，针对完整性与机密性任务评估GIF。即使没有下游脱敏器，GIF也实现近乎完美的召回率，优于基于注意力的基线。结合轻量级基于LLM的脱敏器后，其F1值匹配或超越直接以LLM作为评判器的基线（如GPT-5.5 xhigh推理），同时令牌成本降低高达81倍。使用小型替代模型检测到的GIF流可迁移至更大的先进模型及其他模型家族（即便替代模型小至200倍），这表明无需梯度访问即可进行黑盒部署。

## Abstract
Large language models increasingly mediate interactions between sensitive data, untrusted inputs, and privileged actions in agentic systems, creating security and privacy risks. These range from prompt injections that manipulate downstream tool use to leakage of confidential information through model outputs. Recent Information Flow Control (IFC)-based defenses show promise but lack a principled semantic foundation for reasoning about information flow through the model itself. Since any input token may influence any output token in an autoregressive LLM, existing approaches suffer from severe taint explosion.   We present Geometric Information Flow (GIF), a semantic framework for tracking information flow from input tokens to outputs. GIF uses the LLM Jacobian and local output geometry to upper-bound the Shannon mutual information between perturbed input spans and model outputs, yielding a scalable measure computable on large models via automatic differentiation and low-rank approximation. Unlike attention-based or correlational attribution heuristics, GIF satisfies local geometric soundness, and we provide a fully mechanized Lean 4 proof that it upper-bounds the true information flow induced by a given prompt under local regularity assumptions.   We evaluate GIF on integrity and confidentiality tasks across multiple prompt-injection and privacy-leakage benchmarks. GIF achieves near-perfect recall even without a downstream declassifier, outperforming attention-based baselines. Combined with lightweight LLM-based declassifiers, it matches or exceeds the F1 of direct LLM-as-judge baselines such as GPT-5.5 xhigh reasoning while using up to 81x lower token cost. GIF flows detected with small surrogate models transfer to larger state-of-the-art models and other model families, even when the surrogate is up to 200x smaller, suggesting black-box deployment without gradient access.

## 2. 方法论：核心思想、关键技术细节

- **核心思想**：将信息流定义为输入令牌嵌入扰动与模型输出之间的**香农互信息**。但精确计算不可行，因此通过局部线性近似构造一个**高斯替代信道**，其容量（capacity）作为真实互信息的上界。
- **关键定义**：
  - 对于固定提示x、令牌跨度T和输出位置t*，定义GIF为局部扰动UT ∼ N（0, τI）与输出Y之间的互信息I(UT; Y)。其中UT为对跨度T嵌入的随机扰动，Y为下一令牌分布采样的输出。
  - 利用LLM的雅可比矩阵JT(x)和输出头的Fisher信息矩阵Fsm，定义Fisher拉回矩阵MT = JT^T W^T Fsm W JT，它刻画了输入扰动在输出分布上引起的二阶KL散度变化。
  - 构造高斯信道：YT = AT UT + N，其中AT = Fsm^{1/2} W JT，N为标准高斯噪声。该信道的容量为Cap_T = 0.5 log det(I + τ MT)，该值与真实信道在局部正则性假设下差距为o(τ)。
- **局部可靠性证明**：通过Lean 4完全形式化证明：在Assumption 3.1（局部二次控制）下，GIF（容量）上界真实信息流，即I(UT; Y) ≤ Cap_T(τ; x) + o(τ)。该证明分为三个步骤：变分约简（用p(x)代替边际分布）、局部平均（利用高斯积分得到τ/2 tr(MT)）、容量紧化（容量与迹相差τ²阶）。
- **高效近似**：
  - 直接计算MT需要形成满雅可比矩阵（维度过大），因此采用Hutch++随机迹估计，只需少量矩阵-向量积（通过前向/反向传播实现）。
  - 对于给定span，计算z^T MT z = z^T ξ，其中ξ = JT^T W^T Fsm W JT z，通过一次反向传播得到。
  - 使用ℓ次探针获得估计cInf_T = (1/ℓ)∑z_i^T ξ_i，满足(1-ε) tr(MT) ≤ cInf_T ≤ (1+ε) tr(MT)的概率至少1-δ，当ℓ ≥ 2+4/(ε√δ)。
- **系统设计**：GIF作为组件嵌入智能体推理流程：①标签组件赋予令牌安全标签；②流追踪组件在生成敏感标记（如高完整性/低机密性令牌）时，使用Hutch++计算GIF分数排名；③解分类器（可基于阈值或LLM）根据排名做出ALLOW/DENY决策。

## 6. 主要结论与发现

1. **GIF作为策略信号表现优异**：无需下游解分类器，GIF在大多数模型和基准上AUROC/AUPRC显著优于RTBAS和GIF⁻，平均比GPT-5.5分别提高11.2%和12.0%，同时归因成本降低9.3倍。
2. **GIF结合LLM解分类器可达或超越全轨迹评判**：使用GIF-3压缩后的证据，Qwen3 8B的F1与GPT-5.5 xhigh相当，但token成本降低81倍；在AgentDojo和MSB上token节省63–73%。
3. **Hutch++低秩近似即可保持排序质量**：在Spearman相关系数>0.95时，秩r=4~8已足够，前50重叠>0.90；进一步增加秩主要改善绝对值精度。
4. **小替代模型可迁移到大模型**：使用Qwen3 0.6B作为分析模型，对Qwen3 32B和GPT-OSS 120B的GIF归因效果与原始模型几乎一致（F1变化≤0.7点），提示可在黑盒场景下通过小模型估算流。
5. **GIF可提供0位流出证明**：在案例中，地址“208 Wicker St”未出现在输出中，GIF准确归因0位，而注意力仍显示9%正例，避免了over-taint。
6. **剪切k饱和性**：PSP@k在k≈30时已达95%增益，说明注意力关键令牌数有限，GIF能有效聚焦。

## 7. 优点

- **理论严谨性**：首次为LLM的信息流度量给出了形式化局部可靠性证明（Lean 4验证），确保了上界不会低估真实流，适合用作安全证书。
- **高效可扩展**：通过随机迹估计，只需少量反向传播即可估计整个输入序列的GIF分数，避免了满雅可比矩阵的计算，实际部署可行。
- **跨模型迁移能力**：小代理模型（0.6B参数）可近似200倍大的模型（120B）的流分布，降低了黑盒场景的使用门槛。
- **细粒度与可行性**：输出令牌级别的归因，可以精确到每个输入token的影响力，而非粗粒度的整段标记；在工具调用、记忆更新等场景均适用。
- **成本优势明显**：相比使用强大LLM（如GPT-5.5）作为全轨迹评判器，GIF驱动下的解分类器能大幅减少输入令牌，降低API成本或推理时延。
- **多场景验证**：完整性（提示注入）和机密性（隐私泄露）两个方向均评估，且结果一致。

## 8. 不足与局限

- **局部假设依赖**：理论基于Assumption 3.1（局部二次控制），该假设在非光滑点或远距离扰动下可能不成立。论文未深入验证假设在真实LLM中的有效范围，虽然实验显示在小尺度τ下表现好，但更大扰动可能失效。
- **需要模型内部访问**：GIF计算需要不同类模型（gradient access）以获取雅可比向量积，虽提出可通过小模型替代但仍有依赖。对于完全黑盒API（如GPT-4o无梯度），当前方法无法直接应用（除非使用替代模型估计，但无法保证跨模型一致性）。
- **仅考虑一阶线性近似**：GIF本质上是局部线性化，对于长距离因果依赖或非线性交互（如组合性攻击、跨步推理）可能无法准确捕获。论文未讨论高阶项的影响。
- **基准有限**：只使用了三个基准，且都基于英文Web/工具任务，未包括多语言、多模态、或更复杂的多智能体协作场景。隐私泄露场景仅基于WebArena，可能不覆盖聊天历史中的恶意记忆回溯。
- **安全评估的局限性**：使用benchmark提供的固定标签作为真值，这些标签可能无法完全覆盖真实世界中更微妙的策略违反（如含蓄泄露）。此外，GIF的策略执行依赖于阈值或LLM解分类器，但解分类器本身也可能被欺骗。
- **未评估对抗防御成本**：攻击者可能针对GIF进行对抗性扰动以隐藏源令牌的流（如注入噪声），论文未讨论这种攻击下的鲁棒性。
- **计算开销**：尽管比GPT-5.5便宜，但每次生成敏感令牌时需要额外的Hutch++反向传播，对于高并发服务仍可能带来显著延迟。论文未提供具体延迟数据或与其他轻量级检测方法的效率对比。
